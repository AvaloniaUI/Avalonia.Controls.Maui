# EmbedApp

This application demostrates how to

- Embed Avalonia controls and markup directly in .NET MAUI through the use of `AvaloniaView`
- Create custom Avalonia controls that can be wrapped by either the Avalonia.Controls.Maui `ViewHandler` (To enable it for fully Avalonia driven .NET MAUI applications using Avalonia.Controls.Maui) or the .NET MAUI Native `Microsoft.Maui.Handlers.ViewHandler` for use in native applications.
